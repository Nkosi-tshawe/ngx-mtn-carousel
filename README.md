# MtnCarousel

## Install

```bash
npm install mtn-carousel
```

### Sub Module Import

```typescript
import { LightCarouselModule } from "light-carousel";

@NgModule({
  imports: [
    // ...

    LightCarouselModule,
  ],
})
```

### Usage

```html
<mtn-light-carousel configOptions="options">
  <h1>Title</h1>
</mtn-light-carousel>
```
